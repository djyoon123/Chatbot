# 인사관리 FAQ 챗봇 (RAG + 3-에이전트 하네스)

인사규정·근로기준법 문서를 지식베이스로 삼아, 임직원의 **근태·공수·복지** 등 인사 질문에
**근거 조문·출처·검증 리포트**를 함께 제공하는 RAG 챗봇입니다.

```
질문
 └─▶ [검색 에이전트]  인사규정·근로기준법 조항 top-4 검색 (오프라인)
      └─▶ [답변 에이전트]  근거 조항만 사용 + CoT 자기점검
           └─▶ [검증 에이전트]  조항 매칭 검증 리포트
                └─▶ 사람: 검증 리포트 확인 후 신뢰 여부 판단 (HITL)
```

- **모델**: OpenAI `gpt-4o` (Chat Completions API)
- **검색**: 외부 임베딩 불필요. 문자 바이그램 + 토큰 가중 오프라인 검색
- **하네스**: 검색→답변→검증 3-에이전트 파이프라인 (환각 이중 차단)
- **오프라인**: API 키가 없어도 `faq.json`(검증된 Q&A) 매칭으로 즉답 + 관련 조문 안내

## 구성 파일

| 파일 | 역할 |
|------|------|
| `chatbot.py` | 메인 CLI. 대화 루프 + 모드 전환 |
| `retriever.py` | 인사규정·근로기준법 로드·청킹·검색 |
| `harness.py` | 3-에이전트 파이프라인 (OpenAI 호출) |
| `llm.py` | OpenAI 클라이언트 공급자 (SDK 우선, 표준 라이브러리 폴백) |
| `server.py` | 웹 UI (브라우저 채팅, 표준 라이브러리만) |
| `build_law.py` | 법령 원문(.txt, CP949) → 조문 단위 마크다운 변환 |
| `build_demo.py` | 팀원 공유용 단일 HTML 데모 생성 |
| `faq.json` | 인사·근로기준법 FAQ 데이터셋 |
| `requirements.txt` | 의존성 (`openai`, 선택사항) |

> 지식베이스: `../인사/`, `../법령/` 의 `.md` 파일. (재무·전결규정은 범위 외)
> 규정·법령을 수정하면 재시작 없이 자동 반영됩니다.
>
> **근로기준법**은 상위 법령이므로 함께 인덱싱됩니다. 사내규정이 근로기준법보다
> 근로자에게 불리하면 근로기준법이 우선합니다(근로기준법 제3조). 법령 원문이
> 갱신되면 `python build_law.py`로 `../법령/근로기준법.md`를 다시 생성하세요.
>
> 상세 명세는 프로젝트 루트의 [SPEC_v2.md](../SPEC_v2.md) 참조.

## 설치 및 실행 (Windows)

### 1) 의존성 설치
```powershell
cd "$env:USERPROFILE\Desktop\A1조_FAQ 챗봇\app"
pip install -r requirements.txt
```
> `openai` SDK는 **선택사항**입니다. 설치가 어려운 환경(예: 빌드 휠이 아직 없는
> 베타 Python)에서는 `llm.py`의 표준 라이브러리(urllib) 폴백이 자동으로 동작해
> 동일한 OpenAI API를 호출합니다. (SDK가 설치돼 있으면 SDK를 우선 사용)
>
> 참고: 이 PC처럼 `python` 명령이 Microsoft Store 스텁으로 연결돼 동작하지 않으면
> `py`(Python 런처)를 대신 사용하세요. 예) `py chatbot.py "연차휴가 며칠 발생해?"`

### 2) API 키 설정
```powershell
$env:OPENAI_API_KEY = "sk-..."
```

### 3) 실행

```powershell
# 대화 모드 (하네스, 기본값)
python chatbot.py

# 단발 질문 — 하네스 (검색→답변→검증)
python chatbot.py "연차휴가 며칠 발생해?"

# 단발 질문 — 단순 모드 (답변만, 빠름)
python chatbot.py --simple "법정 근로시간 얼마야?"

# 오프라인 모드 (API 키 없이 동작 — FAQ 기반 답변 + 조문 안내)
python chatbot.py --offline "연차휴가 며칠 발생해?"

# 검색기 단독 점검
python retriever.py "출산전후휴가는 며칠인가요?"
```

### 4) 웹 UI (브라우저에서 사용)

```powershell
py server.py
```
- 브라우저가 자동으로 `http://localhost:8000` 을 엽니다 (안 열리면 직접 접속).
- **API 키 없이도** 오프라인 답변(FAQ 매칭 + 조문 안내)이 동작합니다.
- API 키가 있으면 자동으로 하네스(답변+검증) 모드로 전환됩니다.
- 추가 설치 불필요(Python 표준 라이브러리만 사용). 종료는 터미널에서 `Ctrl+C`.

### 5) 팀원 공유용 단일 HTML 생성

```powershell
py build_demo.py
```
- `../인사관리_FAQ챗봇_공유.html` 1개 파일이 생성됩니다.
- 서버·설치 없이 더블클릭으로 열리는 자체완결 데모 (FAQ·규정 임베드).


## 동작 원리 (RAG + 하네스)

1. **청킹** — 규정 `.md`를 `##`/`###` 헤딩 기준으로 조문 단위로 분할
2. **검색** — 바이그램 중첩(50%) + 토큰 중첩(50%) + 조문 제목 가산점으로 top-4 선택
3. **CoT 자기점검** — 답변 에이전트가 작성 전 근거 매핑 → 1:1 대조 → 없으면 `(확인 필요)` 표기
4. **검증** — 검증 에이전트가 조항 매칭표 + 종합 평가로 인용 정확성 확인
5. **프롬프트 캐싱** — OpenAI가 동일 프리픽스를 자동 캐싱해 반복 호출 비용을 절감
